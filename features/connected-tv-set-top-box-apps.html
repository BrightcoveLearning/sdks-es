Introducción

Los televisores conectados continúan creciendo en uso y popularidad. Con Brightcove, puede llegar al público en sus salas de estar en televisores de pantalla grande con contenido de alta definición.

Los SDK de Native Player brindan integraciones perfectas para controles de IU personalizados, publicidad, protección de contenido (DRM), reproducción en vivo, análisis de Brightcove y métricas de QoE. Para obtener detalles sobre estas funciones, consulte la Funciones del SDK nativo de Brightcove documento.

Brightcove le permite crear aplicaciones de TV atractivas con las siguientes plataformas:

  • SDK de reproductor nativo para tvOS
  • SDK de reproductor nativo para Android
  • Tu propia plataforma

SDK de reproductor nativo para tvOS

Con Brightcove Native Player SDK para tvOS, puede crear experiencias de video atractivas para la última generación de Apple TV. El SDK nativo le permite crear aplicaciones centradas en video para Apple TV utilizando contenido de su biblioteca de Video Cloud, y es compatible con FairPlay Streaming (FPS) para la entrega segura de su contenido premium.

La SSAI básico con la aplicación de muestra tvOS lo ayudará a comenzar proporcionando una base para su aplicación de TV.

 

 

Esta aplicación de muestra le permite incluir la unión de anuncios del lado del servidor para incrustar anuncios en su contenido de video, proporcionando una transmisión continua para la reproducción.

   

Si es un desarrollador que busca más detalles, lea el SDK de reproductor nativo de Brightcove para tvOS documentación técnica.

SDK de reproductor nativo para Android

Amplíe el alcance de su audiencia aprovechando los flujos de trabajo basados en Brightcove. Con Brightcove Native Player SDK para Android, puede crear aplicaciones centradas en video para televisores basados en Android y Fire TV de Amazon.

El SDK nativo para Android le permite aprovechar su contenido de alta definición de su biblioteca de Video Cloud y es compatible con Widevine Modular para la entrega segura de su contenido premium.

La Aplicación básica de muestra de Android lo ayudará a comenzar proporcionando una base para su aplicación de TV.

 

 

Cuando esté listo para introducir controles personalizados en su aplicación, busque la Aplicación de muestra controlada con estilo para ayuda. En este ejemplo, puede ver cómo puede agregar un icono de configuración para controlar la visualización de los subtítulos.

 

 

Una vez que active los subtítulos, los verá aparecer cerca de la parte inferior del video. Esto supone que ha asociado un archivo de subtítulos con su video.

 

 

Si es un desarrollador que busca más detalles, lea el SDK de reproductor nativo de Brightcove para Android documentación técnica.

Tu propia plataforma

Si desea desarrollar aplicaciones de televisión inteligente en su plataforma ganada, utilice las API de Brightcove para integrarlas con su biblioteca de Video Cloud. Las API siguientes son las mismas que utilizan Brightcove Player y los SDK nativos de Brightcove.

API de reproducción

Utilice la API de reproducción para recuperar metadatos de listas de reproducción y vídeos de su biblioteca de Video Cloud. Esta solicitud también devuelve la URL de reproducción actual. Para obtener más detalles, consulte lo siguiente:

  • Descripción general de la API de reproducción

API de recopilación de datos

Utilice la API de recopilación de datos para enviar métricas de reproducción a Brightcove Analytics. Tenga en cuenta que esto se hace automáticamente cuando utiliza Brightcove Player o los SDK nativos. Para obtener más detalles, consulte lo siguiente:

  • Descripción general de la API de recopilación de datos
