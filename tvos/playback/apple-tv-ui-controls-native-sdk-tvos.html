    Controles de interfaz de usuario de Apple TV con el SDK nativo para tvOS

    En este tema, aprenderá acerca de los controles de reproducción proporcionados por Brightcove Player SDK para tvOS.

    Resumen

    Al crear aplicaciones de Apple TV con el SDK nativo de Brightcove para tvOS, obtendrá un conjunto de controles de reproducción para permitir a los usuarios navegar por el contenido de vídeo, cambiar la relación de aspecto del vídeo, obtener información de vídeo y administrar subtítulos y ajustes de audio.

    Los controles proporcionados por el SDK nativo de Brightcove para tvOS incluyen los siguientes:

    • Barra de progreso que también puede mostrar la hora del reloj
    • Barra de pestañas superior para información de vídeo, subtítulos y audio
    • Posibilidad de agregar sus propias vistas y controles personalizados en la barra de pestañas superior

    Los controles y gestos utilizados serán familiares para los usuarios, ya que son similares a los que se encuentran en la interfaz AVKit.

    Con la inserción de anuncios en el lado del servidor (SSAI) de Brightcove, los usuarios verán un aspecto personalizado durante la reproducción de anuncios.

    Para obtener información detallada sobre el uso del SDK nativo, consulte el documento Overview for the Brightcove Player SDK for tvOS .

    Versiones admitidas

    Los controles de interfaz de usuario del Apple TV están disponibles con las siguientes versiones:

    Versión del sistema operativo de dispositivo

    tvOS 9+

    Versión nativa del SDK

    Brightcove Native SDK para iOS versión 6.3.0+

    Mando a distancia Siri

    Siri Remote te permite controlar tu Apple TV. Estas son algunas de las formas en que se puede usar:

    • Pídele a Siri que encuentre lo que quiere ver.
    • Usa la superficie táctil para desplazarse por la pantalla, deslizar, resaltar y seleccionar elementos.
    • Pulse los botones de control para reproducir/pausar contenido rápidamente, ajustar el volumen y navegar a las pantallas de inicio y menú.

    Para obtener más información, consulta el documento Usar tu Siri Remote con Apple TV .

    Controles de reproducción

    La reproducción se controla mediante gestos y pulsaciones de botones en el Apple Siri Remote. A continuación se muestran algunas de las acciones que puede usar con su aplicación. Para obtener más información, consulte lo siguiente:

    • Controles de reproductor de TV integrados para tvOS
    • Uso de la interfaz de usuario del reproductor de TV con el SDK de Brightcove Player para tvOS

    Barra de progreso

    Un solo toque en el trackpad Apple Siri Remote mostrará u ocultará la barra de progreso. Cuando la reproducción está en pausa, el desplazamiento panorámico hacia la izquierda y la derecha se desplazará hacia atrás y hacia adelante a través del vídeo.

    Para fregar, el usuario puede hacer clic y mantener pulsado y, a continuación, desplazar hacia la izquierda o hacia la derecha para avanzar o retroceder rápidamente a través de un vídeo.

    Mientras se reproduce el vídeo, el usuario puede hacer doble clic en el cuarto izquierdo del trackpad Siri Remote para retroceder 10 segundos o hacer doble clic en el cuarto derecho del trackpad para avanzar 10 segundos.

    Relación de aspecto de vídeo

    Un doble toque en el trackpad Apple Siri Remote recorre tres relaciones de aspecto de vídeo estándar:

    • Buzón para respetar la relación de aspecto (vídeo completo)
    • Pantalla completa con relación de aspecto respetada (el vídeo puede recortarse)
    • Vista estirada para adaptarse a pantalla completa (relación de aspecto ignorada)

    Barra de pestañas superior

    Si desliza el dedo hacia abajo en el trackpad Apple Siri Remote, aparecerá el menú superior y ocultará la vista de progreso. Una vez visible, deslizar el dedo hacia arriba ocultará el menú superior.

    La barra de pestañas tiene los siguientes elementos predeterminados:

    • Información
    • Subtítulos
    • Audio

    Puede modificar las etiquetas de texto asociadas a cada elemento de la barra de pestañas o crear sus propias vistas personalizadas en la barra de pestañas.

    Información

    Al seleccionar el elemento de menú Info superior se muestra el título, la descripción y la duración asociados con el vídeo cargado actualmente. Estos datos se recuperan de los metadatos del vídeo en la API de reproducción, pero puede configurar su propio texto cuando se cargue el vídeo.

    Subtítulos

    Al seleccionar el elemento del menú superior Subtítulos , se muestran las opciones para mostrar los subtítulos asociados al vídeo. Aquí, puede administrar los subtítulos/subtítulos que se mostrarán durante la reproducción.

    Audio

    Al seleccionar el elemento del menú superior Audio se muestran las opciones de audio disponibles durante la reproducción de vídeo.

    Personalizado

    Con el SDK nativo de Brightcove para tvOS, puede agregar fácilmente sus propias vistas en la barra de pestañas. Las vistas pueden contener todas las vistas o controles UIKit estándar que necesite.

    Publicidad con SSAI

    Además de la reproducción de contenido, hemos añadido un modo de publicidad al utilizar Brightcove Server-Side Ad Insertion (SSAI). Este modo incluye las siguientes características:

    • Capacidad para evitar que el usuario limpie los anuncios anteriores
    • Mostrar comentarios sobre el tiempo y los anuncios que quedan en un salto de anuncios

    DVR en vivo/vivo

    Cuando trabajas con la aplicación Apple TV, el SDK nativo de Brightcove para tvOS admite transmisiones en directo y DVR. DVR Live permite a los usuarios hacer lo siguiente:

    • Usa la barra de progreso para buscar momentos pasados
    • Vuelve al momento actual en vivo para ver contenido en tiempo real

    Implementación de la reproducción

    Para reproducir una transmisión de DVR en directo o en directo desde la aplicación Apple TV, haz lo siguiente:

    1. Comience con el código de ejemplo básico del Apple TV. Abra el proyecto en Xcode y abra el archivo ViewController.swift .

    2. En la createTVPlayerView() función, establezca el playerType a live o liveDVR.

      playerView = BCOVTVPlayerView(options: options)
if (playerView != nil) {
playerView!.frame = self.videoContainerView.bounds
playerView!.playerType = BCOVTVPlayerType.liveDVR;
self.videoContainerView.addSubview(playerView!)
}
    3. Localice la viewDidLoad() función.
    4. La llamada a la createSampleTabBarItemView() función es opcional y le muestra cómo crear una vista personalizada en la barra de pestañas. Para este ejemplo, vamos a comentar la llamada a esta función.
    5. Comente la llamada a la requestContentFromPlaybackService() función.

    6. En la viewDidLoad() función, antes del corchete de cierre, agregue código para hacer lo siguiente:

      override func viewDidLoad()
{
super.viewDidLoad()
// Do any additional setup after loading the view, typically from a nib.

playbackController.isAutoPlay = true

playerView?.playbackController = playbackController
