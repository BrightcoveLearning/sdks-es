Descripción general

Brightcove Native SDK para Android simplifica la integración de la reproducción de video con Brightcove Video Cloud y muchas soluciones populares de publicidad, análisis y DRM.

El SDK nativo también brinda soporte de subtítulos para dispositivos anteriores a KitKat y controles de reproductor fácilmente personalizables con apariencia similar a Brightcove Web Player para brindar una experiencia de usuario consistente en todas las plataformas.
