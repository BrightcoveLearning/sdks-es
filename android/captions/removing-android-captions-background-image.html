Descripción general

Cuando carga un video con subtítulos en una aplicación creada con Brightcove Native Player SDK para Android, aparecerá un botón de subtítulos en el controlador de medios.

Seleccione el botón de subtítulos en el controlador de medios para mostrar un cuadro de diálogo con un Ajustes botón.

Selecciona el Ajustes para mostrar una imagen de fondo dinámica para mostrar subtítulos mientras los está configurando.

El archivo de imagen de fondo de los subtítulos tiene un tamaño de 400 kb. Si elige deshacerse de esta imagen, puede reemplazarla con un archivo de imagen vacío con el mismo nombre.

Quitar imagen de fondo

Siga estos pasos para eliminar la imagen de fondo de los subtítulos predeterminados:

  1. En el SDK nativo de Brightcove para Android , navega hasta el res / drawable-nodpi / caption_background.jpg expediente.
  2. Cree un archivo vacío con el mismo nombre, caption_background.jpg.
  3. Reemplazar el existente caption_background.jpg archivo con su archivo vacío con el mismo nombre.
  4. Vuelva a compilar su proyecto.
