    Brightcove Player SDK para Android: Información anterior del SDK

    En este tema se muestran las versiones anteriores del SDK y del complemento del SDK del Reproductor nativo para Android.
    Los siguientes vínculos de la versión anterior del SDK de Brightcove Native Player para Android son sólo como referencia. Se recomienda utilizar la última versión del SDK junto con las integraciones de plugin.
