Introducción

Pruebe sus aplicaciones de video con los mensajes de registro predeterminados para ayudar con la depuración. Cuando esté listo para publicar su aplicación, puede eliminar estos mensajes de registro.

Eliminar mensajes de registro

Para eliminar los mensajes de registro predeterminados del SDK nativo para Android, siga estos pasos:

  1. En su aplicación, abra el proguard-rules.pro expediente.
     
    En Android Studio, este archivo se crea de forma predeterminada. Si se pregunta dónde está este archivo en nuestras muestras de Brightcove Android, hemos cambiado el nombre de este archivo a proguard-project.txt.
  2. Agregue el siguiente bloque de código para eliminar el registro de su aplicación: 
    -assumenosideeffects class android.util.Log {
  public static *** v(...);
  public static *** d(...);
  public static *** i(...);
  public static *** w(...);
  public static *** e(...);
}
    Para obtener detalles sobre los mensajes de registro, consulte la Clase de registro de Android documentación.
  3. A continuación, abra la aplicación build.gradle expediente.
  4. En el buildTypes sección, agregue el minifyEnabled atributo con un valor de true.
  5. Debajo del código anterior, agregue un proguardFiles atributo de la siguiente manera:
    • Utilizar el getDefaultProguardFile() método para obtener el proguard-android-Optimize.txt archivo del SDK de Android.
    • Referencia el proguard-rules.pro expediente.

    Tu build.gradle El archivo debería verse similar a esto: 
    android {
    compileSdkVersion 25
    buildToolsVersion '25.0.0'

    defaultConfig {
        ...
    }

    buildTypes {
        release {
            minifyEnabled true
            proguardFiles getDefaultProguardFile('proguard-android-optimize.txt'),
            'proguard-rules.pro'
        }
    }
}
Android Studio deshabilita temporalmente la biblioteca de cobertura de código Java (JaCoCo) y ProGuard mientras usa Instant Run. Debido a que Instant Run solo funciona con compilaciones de depuración, esto no afecta su compilación de lanzamiento (https://developer.android.com/studio/run/index.html).

Si desea eliminar los registros de su tipo de compilación de depuración, debe deshabilitar Instant Run.
