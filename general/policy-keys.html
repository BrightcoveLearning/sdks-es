Introducción

La clave de política se usa para recuperar contenido de video y listas de reproducción de su biblioteca de Video Cloud usando la API de reproducción. Dado que la clave de política está configurada en el reproductor, le permite restringir el acceso a su contenido a nivel de jugador. También puede controlar la disponibilidad del contenido a nivel de video utilizando el Módulo multimedia de Video Cloud Studio.

Cuando crea un Brightcove Player, se genera automáticamente una clave de política para cada jugador. Esta clave está asociada con su cuenta de Brightcove y controla qué videos se pueden ver y cuándo se pueden ver, con la condición de que los videos se almacenen en Video Cloud. Por ejemplo, se necesitan claves de política para limitar el acceso en función de:

  • Cuenta
  • Restricciones de dominio, si se utilizan
  • Restricciones geográficas, si se utilizan

Para obtener información más detallada sobre las claves de política, consulte la Descripción general: API de política documento.

Notas:
  • La clave de política solo se usa para controlar videos que se almacenan en Video Cloud. Si está utilizando Brightcove Player independiente, no es necesario acceder ni recuperar la clave de política.
  • Las claves de política generadas para los nuevos reproductores de Brightcove no están habilitadas para la búsqueda de forma predeterminada. Para buscar videos desde la API de reproducción, debe usar la API de políticas para crear una nueva clave de política que esté habilitada para la búsqueda. Para obtener más detalles, consulte la Descripción general de la API de políticas documento.

Obtener la clave de la política

La clave de la política se genera y administra automáticamente por usted. Por lo tanto, en la mayoría de los casos no necesitará solicitarlo ni administrarlo. La forma más sencilla de recuperar su clave de política es hacer lo siguiente:

  1. Abre el JUGADORES módulo y cree un nuevo reproductor o localice el reproductor al que desea agregar el complemento.
  2. Seleccione el enlace del jugador para abrir las propiedades del jugador.

  3. En el menú de navegación de la izquierda, seleccione Editor JSON.

  4. En el código JSON que se muestra, verá la clave de política:

En raras ocasiones, es posible que no haya creado un Brightcove Player, pero aún necesita su clave de política (como desarrollar estrictamente con los SDK de Brightcove Native Player). En este caso, deberá ponerse en contacto con Atención al cliente para obtener su clave de póliza.

Usando claves de política

La clave de política se puede utilizar para autenticar solicitudes a la API de reproducción. La API de reproducción se utiliza en tiempo de ejecución en páginas web o aplicaciones móviles para la gestión general de medios y la integración con otros sistemas. Para obtener más detalles, consulte la API de reproducción Página de inicio.

Claves de política frente a credenciales de API

La clave de política se utiliza para acceder a la API de reproducción, mientras que las credenciales de la API se utilizan para acceder a su contenido en las otras API de Video Cloud.

Claves de política

La clave de política se utiliza para la recuperación del lado del cliente de los activos de la biblioteca de Video Cloud. Aquí están los detalles:

  • Acceda a los datos de su video / lista de reproducción usando la API de reproducción
  • Solo lectura
  • Los datos de su video / lista de reproducción se almacenan en caché en el servidor mediante la API de reproducción
  • Utilizar el API de política para hacer lo siguiente:
    • Cree claves de política. Debe ponerse en contacto con el servicio de atención al cliente para revocar una clave de política.
    • Establezca restricciones de dominio y establezca restricciones geográficas a nivel de política. Los jugadores que utilizan la clave de política hacen cumplir las restricciones.
También puede utilizar el Métodos de catálogo de Brightcove Player para recuperar contenido de su biblioteca de Video Cloud.

Credenciales de API

Las credenciales de autenticación de API (también llamadas credenciales de cliente) permiten que las aplicaciones del lado del servidor administren los activos de la biblioteca de Video Cloud. Aquí están los detalles:

  • Acceda a sus datos mediante las API de Video Cloud. Necesitará las credenciales de cliente para la cuenta y las API que desea utilizar.
  • Leer, insertar, actualizar, eliminar
  • Los datos de tu video / lista de reproducción no se almacenan en caché en el servidor por la API de CMS
  • Usar Interfaz de usuario de autenticación de API de Video Cloud para crear y administrar credenciales de API

Administrar claves de políticas

Puede administrar sus claves de póliza de la siguiente manera:

Obtención

No hay forma de actualizar una clave de política. Tu también puedes obtener la clave de la política desde un reproductor existente en su cuenta de Video Cloud, o puede crear una clave de política utilizando el API de política.

Revocar

Para revocar una clave de política, debe comunicarse Atención al cliente.

Las claves de política NO se revocan cuando se elimina un jugador. Esto significa que si usa una clave de política de un reproductor y luego elimina el reproductor, aún puede seguir usando la clave de política.
