Descripción general

Una sesión de reproducción representa la reproducción de un solo video. La sesión proporciona un único punto de acceso para todo lo relacionado con la experiencia de reproducción de ese video: el video y fuente seleccionado para la reproducción, el jugador dentro de los cuales ocurre la reproducción, y eventos que ocurren durante la sesión de reproducción. Las sesiones de reproducción nunca se reutilizan para varios videos (incluso en colas separadas del mismo video).

Discusión

El AVMediaSelectionGroup de opciones de selección de medios audibles del AVAsset del AVPlayerItem del AVPlayer de la BCOVPlaybackSession actual.

AVMediaSelectionGroup no está clasificado y puede contener opciones que no se pueden reproducir. Utilice el método + jugableMediaSelectionOptionsFromArray: class de AVMediaSeledtionGroup para devolver la lista de opciones de medios reproducibles. – Terminar método requerido

Termina esta sesión de reproducción, lo que indica que está listo para retirar una nueva sesión. Una sesión de reproducción terminada debe descartarse de inmediato.

- (nulo) terminar

Discusión

Termina esta sesión de reproducción, lo que indica que está listo para retirar una nueva sesión. Una sesión de reproducción terminada debe descartarse de inmediato.

En una configuración típica usando un BCOVPlaybackController , no es necesario llamar a este método directamente. Los métodos del controlador son el mecanismo preferido para avanzar a la siguiente sesión de reproducción. Sin embargo, las configuraciones del proveedor de sesiones de reproducción que no utilizan un BCOVPlaybackController Es posible que deba invocar este método para avanzar en la cola.
