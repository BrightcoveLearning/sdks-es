@property (no atómico, copia) NSString * destino

Discusión

Esta propiedad debe ser un URI con una estructura válida y debe contener una autoridad. El valor predeterminado para esta propiedad, si no se reemplaza, será “bcsdk: //” seguido del identificador del paquete.

Por favor refiérase a http://en.wikipedia.org/wiki/URI_scheme # Generic_syntax para obtener más información y ejemplos de sintaxis de URI.

En particular, un destino sin una parte jerárquica (por ejemplo, solo un esquema) se considera inválido, al igual que cualquier valor sin un esquema.

  uniqueIdentifierEnabled método requerido

Esta propiedad alterna la generación de identificadores únicos del lado del cliente. Si está habilitado, el sdk identificará a los únicos que usan el dispositivo’s identificador de proveedor. Si está deshabilitado, el sdk no proporcionará ningún valor de exclusividad y el análisis se basará en heurísticas impulsadas por el lado del servidor para determinar las exclusivas.

@property (no atómico, asignar, captador = isUniqueIdentifierEnabled) BOOL uniqueIdentifierEnabled

Discusión

Esta propiedad alterna la generación de identificadores únicos del lado del cliente. Si está habilitado, el sdk identificará a los únicos que usan el dispositivo’s identificador de proveedor. Si está deshabilitado, el sdk no proporcionará ningún valor de exclusividad y el análisis se basará en heurísticas impulsadas por el lado del servidor para determinar las exclusivas.

El valor predeterminado es SÍ.
