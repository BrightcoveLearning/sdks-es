Declarado en BCOVCuePoint.h

Descripción general

Clase de valor que representa un punto de referencia en el ecosistema Brightcove Player SDK. Un punto de referencia se utiliza para especificar un posición en la línea de tiempo de un contenido que se puede usar para cualquier propósito (a menudo para el control programático sobre cuándo deben mostrarse los anuncios). Cuando la reproducción alcanza un punto de referencia, la sesión de reproducción envía un evento en su cuePoints señal que incluye el valor BCOVCuePoint.

– actualizar: método requerido

Devuelve una versión modificada de este punto de referencia. Debido a que los objetos BCOVCuePoint son inmutables, se debe crear un BCOVCuePoint completamente nuevo incluso si solo se necesita un cambio. Por lo tanto, este método proporciona una forma conveniente de obtener una copia mutable temporal de esta instancia y modificarla (dentro de updateBlock).

- (tipo de instancia) actualización: (void (^) (id <BCOVMutableCuePoint> mutableCuePoint))updateBlock

Parámetros

updateBlock

Un bloque al que se le pasa una copia mutable de este punto de referencia.

Valor devuelto

La copia de este punto de referencia modificado por updateBlock.

Discusión

Debido a que los objetos BCOVCuePoint son inmutables, se debe crear un BCOVCuePoint completamente nuevo incluso si solo se necesita un cambio. No intente guardar una referencia al punto de referencia mutable pasado a updateBlock , ya que el comportamiento de hacer esto no está definido. En su lugar, realice las modificaciones necesarias en el bloque y luego capture el objeto de punto de referencia devuelto por este método.
