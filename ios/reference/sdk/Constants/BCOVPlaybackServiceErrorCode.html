BCOVPlaybackServiceErrorCodeConnectionError

Código de error que indica que hubo un error al conectarse a la API de reproducción. Verifique la clave error.userInfo NSUnderlyingErrorKey para ver el error de conexión real.

Declarado en BCOVPlaybackService.h.

BCOVPlaybackServiceErrorCodeJSONDeserializationError

Código de error que indica que hubo un error al analizar la respuesta como JSON. Si la solicitud devolvió algún dato, ese NSData sin procesar se incluirá en userInfo mediante la clave kBCOVPlaybackServiceErrorKeyRawResponseData. Verifique la clave NSUnderlyingErrorKey error.userInfo para ver el error de deserialización JSON real.

Declarado en BCOVPlaybackService.h.

BCOVPlaybackServiceErrorCodeAPIError

Código de error que indica que la API devolvió un error. Verifique la clave error.userInfo kBCOVPlaybackServiceErrorKeyAPIErrors para un NSArray de errores devueltos por la API y la clave kBCOVPlaybackServiceErrorKeyAPIHTTPStatusCode para encontrar el código de estado devuelto.
