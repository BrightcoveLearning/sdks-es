Descripción general

Una subclase de UISlider que proporciona un “amortiguado” seguimiento con marcadores para publicidad o uso genérico, más un “modo de publicidad” utilizado para mostrar el progreso de un anuncio.

– addMarkerAt: duration: isAd: image:

Agregue un marcador que se mostrará en el control deslizante en una ubicación específica.

- (vacío) addMarkerAt: (doble)posición duración: (doble)duración isAd: (BOOL)isAd imagen: (UIImage *)imagen

Parámetros

posición

La ubicación (en segundos) en el control deslizante donde se debe colocar el marcador. Ver el “duración” propiedad para detalles relacionados.
duración

los duración representado por la marca de verificación; se puede establecer en cero para dibujar el marcador de tamaño más pequeño (2 píxeles de ancho).
isAd

SÍ indica que este marcador representa un anuncio (dibujado en amarillo de forma predeterminada). NO indica que se trata de un marcador genérico (dibujado en blanco de forma predeterminada).
imagen

Una imagen opcional que se dibujará en lugar del indicador de marcador predeterminado. Se puede establecer en cero.

Discusión

Agregue un marcador que se mostrará en el control deslizante en una ubicación específica.

– setCustomMinimumTrackImage: forState:

Configure una imagen de pista mínima personalizada para usar en el modo normal (sin publicidad).

- (void) setCustomMinimumTrackImage: (UIImage *)imagen forState: (UIControlState)estado

Parámetros

imagen

La imagen que se utilizará en el extremo mínimo de la pista (normalmente el lado izquierdo del pulgar). La imagen debe ser un UIImage redimensionable, ‘trackHeight’ píxeles de alto. Cuando se configura una imagen personalizada, los marcadores ya no se mostrarán. Ajústelo a cero para restaurar las imágenes de pista predeterminadas.
estado

El UIControlState para el que estará activa esta imagen.

Discusión

Configure una imagen de pista mínima personalizada para usar en el modo normal (sin publicidad).
