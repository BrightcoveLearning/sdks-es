Descripción general

BCOVTVInfoTabBarItemView Clase para el “Información” panel de barra de pestañas La vista predeterminada muestra información básica extraída de la respuesta de la API de reproducción. Puede anular esta información manualmente configurando las etiquetas de texto y la vista de imagen cuando tabBarItemViewDidAdvanceToPlaybackSession: se llama.
