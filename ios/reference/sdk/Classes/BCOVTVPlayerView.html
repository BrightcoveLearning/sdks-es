– showView:

Muestra un conjunto específico de vistas en la interfaz de usuario del reproductor de TV.

viewType

Pase BCOVTVShowViewTypeNone para ocultar todas las vistas de control. Pase BCOVTVShowViewTypeControls para mostrar la vista de progreso y sus controles relacionados. Pase BCOVTVShowViewTypeSettings para mostrar la barra de pestañas superior y sus vistas relacionadas.

Discusión
