Descripción general

La configuración global contiene propiedades que se utilizan en todo el SDK de Native Player. Si su aplicación requiere una configuración personalizada, establezca las propiedades necesarias de la Configuración global ANTES de usar cualquiera de los servicios del Native Player SDK.

  chinaProxyDomain

Establece un nombre de dominio completo y personalizado (FQDN) para los servicios de recopilación de análisis, métricas y API de catálogo. Este método se utilizará específicamente para la entrega de contenido confiable en China y requiere arreglos de cuenta particulares. Establecer esta propiedad en cualquier otra situación resultará en un comportamiento desconocido.
