Hereda de NSObject
Declarado en BCOVPlaybackService.h

Descripción general

La clase BCOVPlaybackService proporciona métodos asincrónicos para recuperar información sobre videos de la API de reproducción de Brightcove.

  sharedURLSession

NSURLSession compartida en todas las llamadas de red al BCOVPlaybackService. Este objeto se crea cuando se inicializa el servicio de reproducción. Si desea utilizar su propia NSURLSession, puede configurarla aquí. Si se establece en nil, se volverá a crear la NSURLSession predeterminada.

@property (no atómico, fuerte) NSURLSession * sharedURLSession

– findPlaylistWithPlaylistID: parámetros: finalización:

Recupera un BCOVPlaylist del servicio Playback API por su ID de lista de reproducción en una cola de fondo.

- (vacío) findPlaylistWithPlaylistID: (NSString *)playlistID parámetros: (NSDictionary *)parámetros finalización: (void (^) (BCOVPlaylist * playlist, NSDictionary * jsonResponse, NSError * error))CompletionHandler

Parámetros

playlistID

El ID de la lista de reproducción que se va a buscar.
parámetros

Parámetros de consulta NSString adicionales para agregar a las solicitudes de la API de reproducción. Estos valores anularán los valores predeterminados si entran en conflicto. Puede utilizar los parámetros kBCOVPlaybackServiceParameterKeyOffset y kBCOVPlaybackServiceParameterKeyLimit. Ver “Paginación de lista de reproducción” encima.
CompletionHandler

bloque que se invocará cuando finalice la solicitud. La ejecución de deploymentHandler ocurrirá en el hilo principal.

Discusión

Recupera un BCOVPlaylist del servicio Playback API por su ID de lista de reproducción en una cola de fondo.

Si CompletionHandler proporciona un NSError, el BCOVPlaylist será nulo.
