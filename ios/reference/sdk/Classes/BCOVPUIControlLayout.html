Descripción general

BCOVPUIControlLayout El objeto BCOVPUIControlLayout se crea con un conjunto de controles integrados en BCOVPUILayoutView objetos con algunos parámetros de personalización. El objeto BCOVPUIControlLayout se instala en un BCOVPUILayoutView para mostrar en una vista de jugador.

  standardLayoutItems

standardLayoutItems es la matriz que contiene una o más submatrices de controles que se presentan cuando el ancho de la vista del jugador es mayor que compactLayoutMaximumWidth. Cada submatriz contiene una lista de BCOVPUILayoutView objetos que se presentarán en un BCOVPUILayoutView. Cada submatriz representa una barra de control en la vista del jugador. standardLayoutItems normalmente consta de un único subarreglo que representa una única fila de controles.

  compactLayoutItems

compactLayoutItems es la matriz que contiene una o más submatrices de controles que se presentan cuando el ancho de la vista del reproductor es menor que compactLayoutMaximumWidth. Cada submatriz contiene una lista de BCOVPUILayoutView objetos que se presentarán en un BCOVPUILayoutView. Cada submatriz representa una barra de control en la vista del jugador. compactLayoutItems normalmente consta de dos subarreglos que representan dos filas de controles, donde cada fila tiene menos controles para ajustarse mejor al ancho de la pantalla de un iPhone.

– initWithStandardControls: compactControls:

Inicializa un diseño BCOVPUIControlLayout con la configuración predeterminada.

- (tipo de instancia) initWithStandardControls: (NSArray *)standardLayoutLines compactControls: (NSArray *)compactLayoutLines

Parámetros

standardLayoutLines

Una matriz de matrices, donde cada submatriz corresponde a una fila de controles en la vista de control del reproductor. Cada una de estas submatrices es una lista de objetos BCOVPUILayoutView utilizados en el diseño de ancho estándar. Cada vista de diseño normalmente contiene un único elemento de control y se puede crear con [BCOVPUIBasicControlView layoutViewWithControlFromTag: ancho: elasticidad:]
compactLayoutLines

Una matriz de matrices, donde cada submatriz corresponde a una fila de controles en la vista de control del reproductor. Cada una de estas submatrices es una lista de BCOVPUILayoutView objetos utilizados en el diseño de ancho compacto (normalmente, orientación vertical en un iPhone).

Valor devuelto

Una instancia de BCOVPUIControlLayout inicializada con los controles especificados.

Discusión

Inicializa un diseño BCOVPUIControlLayout con la configuración predeterminada.
