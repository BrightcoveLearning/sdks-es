Descripción general

Player SDK Manager es una instancia única que actúa como fábrica, registro central y centro de diagnóstico en Brightcove Player SDK para iOS. Después de obtener una instancia de Manager, úsela para obtener instancias de otros objetos del ecosistema Player SDK.

Agrega métodos a BCOVPlayerSDKManager para crear reproductores FairPlay y proveedores de sesiones.

@constante kBCOVSSTextTracksKeySourceTypeM3U8URL

Este par clave / valor solo es necesario si la URL de origen se refiere a un archivo M3U8, pero no utiliza un “.m3u8” extensión.

  ID de sesión

sessionID es un identificador único asociado con esta instancia de la aplicación. Este identificador se envía junto con otros datos analíticos a los servidores de métricas de Brightcove.

@property (no atómico, de solo lectura, fuerte) NSString * sessionID

Discusión

sessionID es un identificador único asociado con esta instancia de la aplicación. Este identificador se envía junto con otros datos analíticos a los servidores de métricas de Brightcove.

El sessionID puede ser útil cuando se intenta localizar problemas con problemas de reproducción particulares. Por ejemplo, si detecta problemas de reproducción, puede enviar el ID de sesión y el ID de video a su propio servidor de seguimiento de errores y luego informar esos dos valores a los ingenieros de servicio de Brightcove para rastrear el origen del problema en los registros del servidor.
