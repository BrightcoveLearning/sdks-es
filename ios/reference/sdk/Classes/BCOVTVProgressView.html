Descripción general

Una subclase de UIView que proporciona una “amortiguado” seguimiento con marcadores para publicidad o uso genérico, más un “modo de publicidad” utilizado para mostrar Progreso de anuncios.

– addMarkerAt: duration: isAd: color:

Agregue un marcador que se mostrará en el Progreso ver en una ubicación específica.

- (vacío) addMarkerAt: (doble)posición duración: (doble)duración isAd: (BOOL)isAd color: (UIColor *)color

Parámetros

posición

La ubicación (en segundos) en el Progreso ver dónde debe colocarse el marcador. Ver el “duración” propiedad para detalles relacionados.
duración

los duración en segundos representados por el marcador; se puede establecer en cero para dibujar el marcador de tamaño más pequeño (un círculo).
isAd

SÍ indica que este marcador representa un anuncio. NO indica que se trata de un marcador genérico (dibujado en blanco de forma predeterminada). Por lo general, debe aprobar NO para este valor, ya que los marcadores de anuncios se eliminan y se vuelven a agregar cada vez que avanza a una nueva sesión.
color

Un UIColor que se utilizará para dibujar el marcador. Se puede establecer en cero.

Discusión

Agregue un marcador que se mostrará en el Progreso ver en una ubicación específica.
