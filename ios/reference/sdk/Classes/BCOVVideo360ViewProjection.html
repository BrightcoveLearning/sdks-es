Desplazamiento horizontal del video 360 renderizado en la vista. Cuando se utiliza una vista de gafas de realidad virtual de pantalla dividida, el desplazamiento x se aplica a la vista izquierda y el negativo del desplazamiento se aplica a la vista derecha. Esto se puede usar para ajustar la vista para adaptarse a diferentes tamaños de dispositivos. El valor predeterminado es 0.0.
