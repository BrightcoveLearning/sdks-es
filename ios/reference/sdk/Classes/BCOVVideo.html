  jugable fuera de línea

Determina si se puede reproducir una versión completa de un AVAsset sin una conexión de red.

@property (no atómico, solo lectura) BOOL jugable Fuera de línea

Valor devuelto

Devuelve SÍ si esta instancia se refiere a un desconectado video reproducible desde el dispositivo’s almacenamiento local sin una conexión de red.

Devuelve NO si la descarga del video no se ha completado o si el video se ha purgado y debe volver a descargarse. Esto sucede automáticamente si iOS necesita recuperar espacio de almacenamiento para el correcto funcionamiento del dispositivo.

En algunas versiones de iOS jugable fuera de línea puede devolver NO si el desconectado el video ya está cargado en una instancia de AVPlayer. Vocación replaceCurrentItemWithPlayerItem en la instancia actual de AVPlayer con un nulo El valor antes de la verificación debería dar como resultado el valor esperado de VERDADERO que se devuelva.

Discusión

Consulte esta propiedad antes de presentar o intentar reproducir una desconectado activo para determinar si está disponible para su uso. Un valor de retorno verdadero indica que una versión completa de este activo está disponible para desconectado reproducción.

Leer esta propiedad equivale a leer la propiedad jugableOffline de AVAssetCache de un AVAsset.
