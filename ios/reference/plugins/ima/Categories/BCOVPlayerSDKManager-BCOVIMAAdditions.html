Referencia de categoría de BCOVPlayerSDKManager (BCOVIMAAdditions)

Declarado en BCOVIMAComponent.h

Descripción general

Se agregaron métodos de categoría a BCOVPlayerSDKManager para admitir IMA.

– createIMAPlaybackControllerWithSettings: adsRenderingSettings: adsRequestPolicy: adContainer: viewController: companionSlots: viewStrategy:

Crea y devuelve un nuevo controlador de reproducción con el objeto IMASettings, el objeto IMAAdsRenderingSettings, la estrategia de visualización y el contenedor de anuncios especificados. El controlador de reproducción devuelto se configurará con un proveedor de sesión IMA.

- (id <BCOVPlaybackController>) createIMAPlaybackControllerWithSettings: (IMASettings *)ajustes adsRenderingSettings: (IMAAdsRenderingSettings *)adsRenderingSettings adsRequestPolicy: (BCOVIMAAdsRequestPolicy *)adsRequestPolicy adContainer: (UIView *)adContainer viewController: (UIViewController *)adContainerViewController companionSlots: (NSArray *)companionSlots viewStrategy: (BCOVPlaybackControllerViewStrategy)estrategia

Parámetros

ajustes

Un IMASettings que se utilizará para configurar cualquier objeto IMAAdsLoader utilizado por el controlador de reproducción devuelto.
adsRenderingSettings

Un IMAAdsRenderingSettings que se utilizará para configurar cualquier objeto IMAAdsManager utilizado por el controlador de reproducción devuelto.
adsRequestPolicy

BCOVIMAAdsRequestPolicy instancia para generar IMAAdsRequests para su uso por una determinada sesión de reproducción de entrada.
adContainer

la vista en la que se mostrará el anuncio y la interfaz de usuario de información del anuncio.
adContainerViewController

El controlador de vista que contiene el contenedor de anuncios. Si no se proporciona aquí, debe configurarse en IMAAdDisplayContainer antes de realizar una solicitud de anuncios o transmisión.
companionSlots

la lista de instancias de IMACompanionAdSlot. Puede ser nulo o vacío.
estrategia

Una estrategia de vista que determina la vista del controlador de reproducción devuelto.

Valor devuelto

Un nuevo controlador de reproducción con los parámetros especificados.

Discusión

Crea y devuelve un nuevo controlador de reproducción con el objeto IMASettings, el objeto IMAAdsRenderingSettings, la estrategia de visualización y el contenedor de anuncios especificados. El controlador de reproducción devuelto se configurará con un proveedor de sesión IMA.
