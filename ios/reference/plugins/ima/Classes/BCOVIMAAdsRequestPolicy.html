Declarado en BCOVIMAAdsRequestPolicy.h

Descripción general

Objeto de política para generar IMAAdsRequests para su uso por una sesión de reproducción de entrada determinada.

+ videoPropertiesVMAPAdTagUrlAdsRequestPolicy

Devuelve una política de solicitud de anuncios que verifica el objeto BCOVVideo en cada sesión de reproducción para la URL de la etiqueta de anuncio VMAP para solicitar. Si el objeto de video’s propiedades contiene una entrada cuya clave es kBCOVIMAAdTag , se supone que el valor de esa entrada es un NSString que contiene la URL de la etiqueta publicitaria VMAP. Esa URL se utilizará para construir una VMAP IMAAdsRequest para esa sesión de reproducción.

+ (tipo de instancia) videoPropertiesVMAPAdTagUrlAdsRequestPolicy

Valor devuelto

Una política de solicitud de anuncios que genera VMAP IMAAdsRequests a partir de la información de cada sesión de reproducción.’s BCOVVideo.properties.

Discusión
