– adecuadoSourceNotFound

Este método se llama cuando no se encuentra una fuente adecuada.

- (vacío) adecuadoSourceNotFound

Discusión

los BCOVGoogleCastManager buscará la versión HTTPS de HLS v3, luego buscará DASH y finalmente una fuente MP4. Si no se encuentran versiones HTTPS, buscará versiones HTTP de los mismos tipos de fuente. Si no se encuentran versiones HTTP, se llamará a este método.

– shouldCastVideo:

Este método se llama después BCOVGoogleCastManager ha recibido una nueva sesión de reproducción.

- (BOOL) shouldCastVideo: (BCOVVideo * _Nonnull)video

Discusión

Cuando BCOVGoogleCastManager recibe un nuevo BCOVVideo de una sesión de reproducción, intentará compararlo con el BCOVVideo anterior, si lo hay, para evitar que la sesión de transmisión se reinicie con el mismo video. Puede usar este método para anular la verificación interna de modo que tenga control sobre si el video actual debe transmitirse o no.
