Descripción general

Esta clase se utiliza para mostrar la interfaz de usuario del anuncio y gestionar anuncios complementarios para BCOVOUX.

La interfaz de usuario incluye un banner que muestra la cuenta regresiva hasta que comienza el contenido y el botón para aprender más y un botón para omitir para los anuncios que se pueden omitir. Este objeto solo agrega la vista del anuncio a la jerarquía cuando se va a reproducir un anuncio y se elimina cuando el anuncio está completo.

Cuando un anuncio contiene complementarios, este objeto intentará hacer coincidir el ancho y el alto de esos complementarios con el ancho y alto de los BCOVOUXCompanionSlots administrados por este objeto. Si no se encuentra ninguna coincidencia, el complementario no se completará. Si se produce una coincidencia, el espacio complementario permanecerá ocupado hasta que otro anuncio con un espacio complementario coincidente ocupe su lugar.

Después de crear este objeto, debe agregarse al controlador de reproducción como consumidor de sesión a través de [BCOVPlaybackController addSessionConsumer:].
