Discusión

Este método implementa el comportamiento de búsqueda predeterminado para el complemento BCOVOUX. Este método debe invocarse en el hilo principal. El receptor ignorará este mensaje cuando haya un anuncio en curso o una llamada anterior a oux_seekToTime: CompletionHandler: todavía se está procesando. Ver también oux_canSeek.

Las reglas son las siguientes:

  • Si busca finalizar el tiempo de contenido lógico, se reproducirán las listas de publicaciones.
  • Si busca sobre una secuencia de anuncios, el anuncio se reproducirá y el cabezal de reproducción se reanudará donde la búsqueda había intentado realizar la búsqueda.
  • Si busca varias secuencias de anuncios, solo se reproducirá la última secuencia de anuncios.
  • Si busca al revés, no se reproducirán anuncios.
  • Si busca anuncios que ya se han reproducido, volverán a reproducirse.

Discusión

Devuelve el tiempo absoluto que corresponde al tiempo de contenido especificado.

Este método se utiliza para obtener un tiempo de transmisión absoluto que corresponde a un tiempo relativo al contenido. Por ejemplo, la transmisión puede incluir una secuencia de anuncios de treinta segundos en kCMTimeZero , después de lo cual comienza el contenido. En este caso, un tiempo de contenido de cero correspondería a un tiempo de transmisión absoluto de treinta segundos, y un tiempo de contenido de 2:00 correspondería a un tiempo de transmisión absoluto de 2:30 (suponiendo que no se produzcan otras secuencias de anuncios antes de esa hora). .

Discusión

Devuelve el tiempo del contenido que corresponde al tiempo absoluto especificado.

Este método se utiliza para obtener un tiempo de transmisión de contenido que corresponde a un tiempo relativo a la carga útil. Por ejemplo, la transmisión puede incluir una secuencia de anuncios de treinta segundos a la 1:30, después de la cual comienza el contenido.

En este caso, un tiempo de contenido de cero correspondería a un tiempo de transmisión absoluto de cero, un tiempo de contenido de 2:00 correspondería a un tiempo de transmisión absoluto de 2:30 y un tiempo de contenido de 1:30 correspondería a cualquier tiempo de transmisión absoluto de 1:30 a 2:00.
