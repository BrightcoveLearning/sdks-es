Descripción general

Implemente este protocolo cuando interactúe con un servidor de licencias FPS personalizado.

– contentIdentifierFromLoadingRequest:

Método opcional. Implemente este método para manejar la recuperación del identificador de contenido de la solicitud de carga. Esto se llama para cada activo. No se ofrece ninguna garantía en cuanto al hilo de que se solicita. Debe devolver el identificador de contenido correspondiente a esta solicitud de carga. Esto normalmente se extrae de la “skd: //…” URL en loadingRequest.request.URL. El identificador de contenido se pasará a [AVAssetResourceLoadingRequest streamingContentKeyRequestDataForApp: contentIdentifier: options: error:] para recuperar el SPC.

- (NSData que acepta valores NULL *) contentIdentifierFromLoadingRequest: (AVAssetResourceLoadingRequest * no nulo)loadingRequest

Parámetros

loadingRequest

La solicitud de carga skd: // realizada para el activo.

Discusión
