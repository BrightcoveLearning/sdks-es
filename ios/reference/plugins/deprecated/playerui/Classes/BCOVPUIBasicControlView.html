+ layoutViewWithControlFromTag: ancho: elasticidad:

Método de fábrica de conveniencia para crear un diseño vista con una subvista determinada por la etiqueta especificada. Las vistas de diseño se agrupan en matrices y se utilizan para crear un BCOVPUIControlLayout ejemplo.

+ (BCOVPUILayoutView *) layoutViewWithControlFromTag: (BCOVPUIViewTag)etiqueta ancho: (CGFloat)ancho elasticidad: (CGFloat)elasticidad

Parámetros

etiqueta

Un valor de etiqueta que indica el control deseado o vacío diseño vista.
ancho

El ancho mínimo del nuevo diseño vista. Pase kBCOVPUILayoutUseDefaultValue para usar el ancho predeterminado para el control específico.
elasticidad

La cantidad relativa que un diseño La vista se puede expandir para llenar el espacio. La elasticidad es relativa a otras diseño puntos de vista.

Valor devuelto

A BCOVPUILayoutView instancia inicializada con una subvista especificada por la etiqueta, el ancho y la elasticidad.

Discusión

Método de fábrica de conveniencia para crear un diseño vista con una subvista determinada por la etiqueta especificada. Las vistas de diseño se agrupan en matrices y se utilizan para crear un BCOVPUIControlLayout ejemplo.

+ createPUIControlItemWithViewTag:

Construye el elemento de control de la interfaz de usuario con la etiqueta especificada. El objeto devuelto puede ser un Botón BCOVPUIB , BCOVPUISlider , UILabel o MPVolumeView según la etiqueta especificada. Este método normalmente solo lo utilizan layoutViewWithControlFromTag: ancho: elasticidad: para crear el control colocado dentro de un BCOVPUILayoutView. Se aplicarán los estilos predeterminados.
