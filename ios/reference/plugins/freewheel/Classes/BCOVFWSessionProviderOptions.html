Descripción general

  waitForAVPlayerDuration

Si esperar o no a que el elemento AVPlayer informe una duración antes de llamar a adContextPolicy. Cuando se establece en NO, la duración pasa al BCOVFWSessionProviderAdContextPolicy el bloque puede no ser exacto. El valor predeterminado es Sí.

@property (nonatomic, assign, getter = shouldWaitForAVPlayerDuration) BOOL waitForAVPlayerDuration

  preloadSlots

Determina si las tragamonedas están precargadas o no para que estén listas para jugar cuando se golpean. Cuando se establece en SÍ, las ranuras de pre-roll se cargarán lo antes posible. Las tragamonedas midrolls se cargarán X segundos antes de que se juegue la tragamonedas, determinado por timeBeforeAdToPreload. Por defecto es NO.

@property (nonatomic, assign, getter = shouldPreloadSlots) BOOL preloadSlots

Discusión

Determina si las tragamonedas están precargadas o no para que estén listas para jugar cuando se golpean. Cuando se establece en SÍ, las ranuras de pre-roll se cargarán lo antes posible. Las tragamonedas midrolls se cargarán X segundos antes de que se juegue la tragamonedas, determinado por timeBeforeAdToPreload. Por defecto es NO.
