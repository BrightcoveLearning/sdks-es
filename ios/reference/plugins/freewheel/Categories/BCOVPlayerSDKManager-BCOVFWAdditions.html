Referencia de categoría de BCOVPlayerSDKManager (BCOVFWAdditions)

Declarado en BCOVFWComponent.h

Descripción general

Se agregaron métodos de categoría a BCOVPlayerSDKManager para admitir FW.

– createFWPlaybackControllerWithAdContextPolicy: viewStrategy:

Crea un controlador de reproducción con adContextPolicy y viewStrategy dados. Este método utiliza un complemento FreeWheel con su configuración predeterminada. Para personalizar el complemento FreeWheel, use createFWSessionProviderWithAdContextPolicy: upstreamSessionProvider: options.

- (id <BCOVPlaybackController>) createFWPlaybackControllerWithAdContextPolicy: (BCOVFWSessionProviderAdContextPolicy)adContextPolicy viewStrategy: (BCOVPlaybackControllerViewStrategy)estrategia

Parámetros

adContextPolicy

El bloque que construye el FWAdContext que se usará por video.
estrategia

La estrategia de vista para la propiedad playbackController.view.

Valor devuelto

Un controlador de reproducción.
