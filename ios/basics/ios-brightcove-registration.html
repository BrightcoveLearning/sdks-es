Descripción general

Para los clientes de Video Cloud, en la mayoría de los casos, cuando utiliza Brightcove Player SDK para iOS, su aplicación se registra automáticamente. Para los clientes de Brightcove Player, debe enviar su identificación de cuenta desde su aplicación. Vea los detalles a continuación.

Clientes de Video Cloud

Si utiliza el reproductor Brightcove y el Catalog clase incluida en el SDK para recuperar y reproducir videos, su aplicación se registrará en Brightcove automáticamente. Además, los datos analíticos se recopilarán y aparecerán en su Video Cloud Analytics.

Si anula el BCOVVideo class o no utilice el reproductor y el catálogo de Brightcove, debe enviar su ID de editor de Video Cloud a Video Cloud Analytics. Puede hacer esto configurando la analítica account valor. Esto también le permite ver los datos de esta aplicación en Video Cloud Analytics.

static NSString * const kPublisherID = @"<insert valid accountId>"; // for analytics

    _playbackController.analytics.account = kPublisherID;

Para obtener detalles sobre el envío de datos a Video Cloud Analytics, consulte la Uso de Brightcove Analytics guía.

Clientes de Brightcove Player

Como cliente de Brightcove Player, debe registrar sus aplicaciones en Brightcove. Para hacer esto, enviará su ID de cuenta a Brightcove utilizando los análisis account valor.

static NSString * const kPublisherID = @"<insert valid accountId>"; // for analytics

    _playbackController.analytics.account = kPublisherID;
 

Ahora que sabe cómo registrar su aplicación, está listo para aprender a trabajar con su contenido multimedia en el siguiente documento.
