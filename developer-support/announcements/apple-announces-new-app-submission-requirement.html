Requisitos para App Store

Para aprovechar las nuevas funciones proporcionadas por el SDK de iOS, Apple requerirá que todas las aplicaciones nuevas enviadas a la App Store se compilen con el SDK de iOS 12.1 o posterior. Este requisito entra en vigor a partir de Marzo de 2019.

Esto significa que para enviar aplicaciones nuevas, deberá usar Xcode 10.1, que proporciona las herramientas para crear aplicaciones para los últimos SDK, incluidos iOS 12, tvOS 12 y Swift 4.2.

Para obtener más detalles, consulte la Envío de aplicaciones de iOS a la App Store documentación.

Compatibilidad con SDK nativo

Siguiendo el ejemplo de Apple, Brightcove Native SDK para iOS / tvOS v6.3.10 + admite la construcción con Xcode 10.1. Si aún no lo ha hecho, migre a Xcode 10.1 lo antes posible.

El SDK nativo de Brightcove para iOS / tvOS v6.3.5 + ya no admite la construcción con Xcode 8.
