A continuación se muestra una lista de algunas preguntas frecuentes asociadas con los SDK de Brightcove Native Player. ¿Tiene alguna pregunta que no ve aquí? ¡Haznos saber!

Preguntas generales

¿Se admiten los subtítulos ocultos?

Los Player SDK son compatibles con WebVTT en manifiesto (HLS), así como con subtítulos de WebVTT y 608 sobre 708. Para obtener detalles de soporte, consulte el Formatos de video y subtítulos documento.


¿Puedo personalizar la barra de búsqueda?

Si. Si usa los controles de Brightcove Native Player, puede personalizar los elementos, incluida la barra de búsqueda. Para obtener más información, consulte el Controles del reproductor nativo de Brightcove documento.


¿Puedo superponer el video con controles personalizados?

Los Player SDK no te obligan a usar sus controles. Puedes elegir usarlos o no. Si usa los controles de Brightcove Native Player, puede personalizar los elementos. Para obtener más información, consulte el Controles del reproductor nativo de Brightcove documento.


¿Puedo agregar la funcionalidad de video de 10 segundos ff / rew?

Los SDK del reproductor admiten la búsqueda, por lo que puede implementar esta funcionalidad.


¿Es compatible con Chromecast?

Los SDK nativos de Brightcove admiten la transmisión de videos a televisores conectados a Chromecast.

  • Para dispositivos Android, use el Complemento Cast para el SDK nativo para Android.
  • Para dispositivos iOS, use el Complemento Cast para SDK nativo para iOS.

¿Se admiten las listas de reproducción?

Sí, los SDK nativos de Brightcove proporcionan métodos para recuperar un video o una lista de reproducción de la API de reproducción. Para obtener más detalles, consulte lo siguiente:

  • VideoListSampleApp para el SDK nativo para Android
  • Recuperación de activos de Brightcove mediante el servicio de reproducción para el SDK nativo para iOS

Específico de Android

¿Cómo puedo hacer que HLS funcione en Android?

Player SDK de Brightcove para Android proporciona la funcionalidad de reproducción HLS para Android 4.1 y versiones posteriores.


¿Qué pasa con HLSe?

El Player SDK de Brightcove para Android admite la reproducción con HLSe.

específico de iOS

¿Es compatible con AirPlay?

Player SDK para iOS es compatible con AirPlay siempre que no tenga anuncios. AirPlay de Apple le permite transmitir contenido de forma inalámbrica desde sus dispositivos Apple.


¿Es compatible el Apple TV?

La última versión del Player SDK para iOS es compatible con el sistema operativo tvOS de Apple TV.
