Características

Los SDK de Brightcove admiten un amplio conjunto de funciones que incluyen las siguientes:

  • Subtítulos
  • Publicidad
  • Analítica
  • Protección de contenido
  • Reproducción en vivo
  • Controles de interfaz de usuario estándar
  • Video de 360 °
  • Reproducción sin conexión

Productos

Diseñados pensando en los desarrolladores de plataformas nativas, los SDK nativos de Brightcove tienen como objetivo minimizar el tiempo de comercialización siguiendo los últimos estándares en el desarrollo de aplicaciones en cada entorno.

Los SDK nativos de Brightcove se crean sobre marcos de reproductores nativos. Esto le permite aprovechar la velocidad, el rendimiento y la flexibilidad del sistema operativo nativo y extenderlo a una experiencia de video completa que es fácil de integrar. Además, nuestros SDK nativos siguen las convenciones establecidas para los patrones de memoria, nomenclatura y diseño para que comenzar sea lo más simple posible.

Los SDK nativos están diseñados para integrarse con los siguientes tipos de dispositivos:

  • Dispositivos móviles
  • TV / decodificadores conectados

Dispositivos móviles

Ofrecemos los siguientes SDK para dispositivos móviles, cada uno especializado para su propio propósito específico:

  • SDK nativo de Brightcove para iOS
  • SDK nativo de Brightcove para Android

SDK nativo de Brightcove para iOS

El SDK nativo de Brightcove para iOS sigue las convenciones modernas de iOS, por lo que si se está integrando en una aplicación Objective-C establecida o en una nueva aplicación Swift, debe sentirse como en casa.

Versiones de iOS compatibles
Para aprovechar las últimas mejoras tecnológicas de Apple, actualmente ofrecemos soporte para:

  • iOS 14
  • iOS 13.7
  • iOS 12.5.3

Con el lanzamiento de iOS 9 y Xcode 7, aprenda a trabajar con una nueva función llamada Seguridad de transporte de aplicaciones (ATS).

Utilizando una combinación de métricas recopiladas internamente y los informes de datos de uso de dispositivos iOS públicos de Apple, Brightcove SDK para iOS tiene como objetivo llegar a la gran mayoría de usuarios.

Si es un desarrollador que busca más detalles, lea el SDK nativo de Brightcove para iOS documentación técnica.

SDK nativo de Brightcove para Android

Brightcove Native SDK para Android tiene como objetivo equilibrar el alcance de la gran mayoría de usuarios mientras se aprovechan las mejoras lanzadas en las últimas versiones de Android.

Versiones de Android compatibles
Al equilibrar este alcance con la capacidad de aprovechar las mejoras lanzadas por Google, actualmente ofrecemos soporte para las siguientes versiones de Android:

  • 10.0
  • 9.0
  • 8.0, 8.1
  • 7.0, 7.1
  • 6.0

Usando una combinación de métricas recopiladas internamente y el público de Google Datos de uso del dispositivo Android informe, Brightcove SDK para Android tiene como objetivo llegar a la gran mayoría de usuarios.

Si es un desarrollador que busca más detalles, lea el SDK nativo de Brightcove para Android documentación técnica.

TV / decodificadores conectados

Ofrecemos los siguientes SDK para integrarlos con decodificadores de TV:

  • SDK nativo de Brightcove para tvOS
  • SDK nativo de Brightcove para Android

SDK nativo de Brightcove para tvOS

Cautive a la audiencia de su sala de estar con aplicaciones centradas en video en la cuarta generación de Apple TV aprovechando el contenido de Video Cloud a través del Brightcove Native SDK para tvOS.

Versiones de tvOS compatibles
Para aprovechar las últimas mejoras tecnológicas de Apple, actualmente ofrecemos soporte para:

  • tvOS 14
  • tvOS 13.4.8
  • tvOS 12.4.1

Aprender más sobre Aplicaciones de TV conectadas usando los SDK nativos.

Revisar la Brightcove Jump Start para Apple TV.

SDK nativo de Brightcove para Android

Expanda el alcance de su contenido de video para incluir una variedad de plataformas de TV inteligente basadas en Android y Fire TV, utilizando Brightcove Native SDK para Android.

Actualmente ofrecemos soporte para:

  • Android TV
  • Fire TV (dispositivos FireOS 5.0+)
